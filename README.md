# NLP-Text-Classifier
Magic Vendor NPC — NLP Intent Classifier A PyTorch BiLSTM neural network that reads open-ended player input and classifies it into 5 intents (buy, sell, haggle, lore, other) for an Unreal Engine NPC. Uses word embeddings for semantic understanding and exports to ONNX for C++ inference via UE's Neural Network Inference (NNI) plugin.
